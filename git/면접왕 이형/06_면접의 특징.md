# 면접의 특징

##### 면접이란

- **일관성**을 본다
- 자기소개서 : 나의 성향/경험
- 직무적성검사의 통계 : 나의 성향
- 면접 : 여러 경험을 증명하는 것



- `면접 이전` : 떨어뜨릴 이유 있는 사람을 거르는 것
- `면접 이후` : 뽑아야 할 이유 있는 사람을 뽑는 것
  - 면접은 평타만 쳐서는 절대 안된다.
  - 필살기를 보여야 한다
    - 내 모습을 정확하게 전달
    - 내 직무상 강점
    - 이 포지션에서 어떤 역할을 할 수 있을지
    - 나의 기여와 공헌
